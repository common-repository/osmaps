Plugin to view free road maps.

== Description ==
Lightweight Plugin to display in your website multiple free road maps.<br>You can have as many maps as you need but only one per page.<br>You can display a "VIEW MAP" button. A single page can have many buttons.<br>To display a Map you can choose between Api javascript Openlayers and Api Leaflet <br> Map tiles by OpenStreetMap. Access key is not required.<br>Map tiles by Mapbox. Access key is required. Mapbox create satellite maps.

= Features =
- Simple Map.
- Complex Map, with Marker and / or PopUp. The popup created with the Api OpenLayers has a green icon with links to the Directions.
- Multiple maps.
- Road maps and satellite maps.
- Displays a "VIEW MAP" button. View the map, full screen, on a new page.

== Demo ==
[View:]( https://osmaps.edilweb.eu )

== Installation ==
Upload the OSMaps plugin to your site and activate it.

== Minimal usage ==
- Configure the Settings -> OSMaps from the admin dashboard. Change the defaultMap options and save.
- Insert the shortcode in an article or in a page. Look the OSMaps administrative page for the shortcodes.
- Done!
